# AdventureGo - Travel App Concept

## Overview

AdventureGo is a personal UI/UX design practice project created in Figma. The project explores a modern travel and adventure booking experience that helps users discover destinations, explore travel experiences, and book memorable trips through an intuitive mobile interface.

---

## Project Type

Personal Practice Project

---

## My Role

UI/UX Designer

---

## Tools Used

- Figma

---

## Features

- Welcome & Onboarding Screen
- Destination Discovery
- Travel Search
- Adventure Details
- Trip Booking
- User Login

---

## Project Goal

The goal of this project was to practice designing a modern travel application with a focus on user-friendly navigation, engaging visual layouts, and an enjoyable booking experience.

- User Interface Design
